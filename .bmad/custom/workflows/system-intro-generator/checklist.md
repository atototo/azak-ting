- [ ] 종목 정보가 정확함 (코드, 이름, 주가)

## 시스템 설명 정확성

- [ ] "AI에게 물어봤다" 같은 잘못된 표현 없음

- [ ] A/B 모델 교체 가능하다는 설명 포함
- [ ] 자동화 프로세스 설명 (10분, 5분, 하루 3회 등)

## 자연스러운 글쓰기

- [ ] 마침표(.) 과다 사용 안 함
- [ ] 불릿 포인트(-) 나열식 최소화
- [ ] 이모지 적절히 사용 (섹션당 0~1개)
- [ ] "여러분", "~하세요" 클리셰 없음
- [ ] 구어체 종결어미 사용 (~거든, ~거야, ~지, ~더라)
- [ ] 짧은 문장 위주
- [ ] 솔직한 감정 표현 포함
