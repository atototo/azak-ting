# Validation Checklist - Stock Blog with SNS

## Structure

- [ ] 블로그 포스트 섹션이 존재하며 완전한 내용을 포함
- [ ] Twitter/X 섹션이 존재하며 280자 제한 준수
- [ ] Instagram 섹션이 존재하며 2200자 제한 준수
- [ ] LinkedIn 섹션이 존재하며 3000자 제한 준수
- [ ] SNS 콘텐츠 활용 가이드 섹션 포함
- [ ] 모든 변수 플레이스홀더({{variable}})가 실제 값으로 치환됨
- [ ] 마크다운 포맷이 올바르게 적용됨

## Content Quality - Blog Post

- [ ] 종목 코드와 종목 이름이 정확히 반영됨
- [ ] AI 분석 리포트 내용이 포함됨
- [ ] 스크린샷 이미지 경로가 올바름
- [ ] 톤&매너 가이드라인 준수 (존댓말 통일, 구어체 혼용)
- [ ] 과도한 마침표(.)나 콜론(:) 사용 없음
- [ ] 불릿 포인트 나열만 하지 않고 자연스러운 글
- [ ] 마크다운 테이블이 적절히 사용됨 (데이터 정리)

## Content Quality - SNS Content

### Twitter/X
- [ ] 280자 이내로 작성됨
- [ ] 핵심 메시지가 명확함
- [ ] 해시태그 2-3개 포함
- [ ] 블로그 링크 공간 확보됨
- [ ] 임팩트 있는 첫 문장
- [ ] 핵심 수치 1-2개 포함

### Instagram
- [ ] 2200자 이내로 작성됨
- [ ] 이모지가 적절히 사용됨 (과하지 않게)
- [ ] 3-5개의 핵심 포인트 포함
- [ ] 해시태그 5-10개 포함
- [ ] 친근하고 시각적인 톤
- [ ] 프로필 링크 안내 포함

### LinkedIn
- [ ] 3000자 이내로 작성됨
- [ ] 전문적이면서 접근하기 쉬운 톤
- [ ] 투자 인사이트와 분석 강조
- [ ] AI 모델 비교 관점 포함 (해당되는 경우)
- [ ] 해시태그 3-5개 포함
- [ ] Call-to-action 명확함

## Platform-Specific Best Practices

- [ ] Twitter/X: 간결하고 임팩트 있음, 즉시성 강조
- [ ] Instagram: 시각적 요소 언급, 캐러셀/스토리 활용 가능
- [ ] LinkedIn: 전문성 유지, 심층 분석 강조

## Completeness

- [ ] blog-post-generator 워크플로우가 성공적으로 실행됨
- [ ] 모든 SNS 플랫폼 콘텐츠가 생성됨
- [ ] 출력 파일이 올바른 경로에 저장됨
- [ ] 종목 정보가 모든 섹션에 일관되게 반영됨
- [ ] 사용 팁과 활용 가이드가 명확함
- [ ] 면책 조항이 포함됨

## Technical Validation

- [ ] 변수 이름이 instructions.md의 <template-output>과 일치
- [ ] 파일 경로가 workflow.yaml 설정과 일치
- [ ] 의존 워크플로우(blog-post-generator) 경로가 올바름
- [ ] 날짜 형식이 올바르게 생성됨
- [ ] 파일명에 특수문자나 공백 없음
