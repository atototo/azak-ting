# Azak 개발 과정 블로그 생성

**목적**: 최근에 개발한 기능의 개발 과정을 블로그로 작성합니다.

**구조**: 문제 상황 → 해결 과정 (코드 포함) → 결과 → 배운 점 → 기술 스택

**특징**:
- 코드 분석 & 스니펫 포함
- 문제-해결 구조 (삽질 경험)
- 화면 캡처 + GIF 자동 생성
- 개발자다운 구어체 톤

---

**Azak 개발 과정 블로그 워크플로우를 실행합니다.**

이 워크플로우는 다음을 수행합니다:
1. 개발 주제 입력
2. 관련 파일 코드 분석
3. Git 히스토리 분석 (선택)
4. 화면 캡처 (스크린샷 + GIF)
5. 블로그 포스트 자동 생성

**사용 예시**:
- 주제: "Celery로 비동기 리포트 시스템 만들기"
- 관련 파일: backend/tasks/reports.py
- GIF URL: /stocks/005930
