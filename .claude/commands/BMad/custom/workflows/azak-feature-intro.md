# Azak 기능 소개 블로그 생성

**목적**: 이미 구현된 기능을 소개하는 블로그를 작성합니다.

**구조**: 기능 개요 → 주요 기능 → 사용 방법 (화면 중심) → 기술 스택 → 향후 계획

**특징**:
- 화면 캡처 중심 + GIF 자동 생성
- 사용 방법 단계별 설명
- 코드는 선택적 (필요시만)
- 문제 상황/해결 과정은 선택적
- 친근한 설명 톤

---

**Azak 기능 소개 블로그 워크플로우를 실행합니다.**

이 워크플로우는 다음을 수행합니다:
1. 기능명 & 설명 입력
2. 주요 기능 정리
3. 화면 캡처 (스크린샷 + GIF)
4. 사용 방법 작성
5. 블로그 포스트 자동 생성

**사용 예시**:
- 기능명: "비동기 리포트 생성 시스템"
- 주요 기능: 백그라운드 처리, Toast 알림, 5초 폴링
